# Our Little Museum 🎞

A private digital gallery built for couples — a space to upload photos, write stories, and keep every shared moment alive.

## What it is

Our Little Museum is a beautifully designed web app that works like a personal scrapbook but smarter. You can upload photos, tag them with a feeling and a date, write a little story for each one, and then browse everything by mood, timeline, or a gallery grid.

No accounts, no servers — everything stays private on your device.

---

## Features

- **Gallery view** — a clean, browsable grid of all your memories
- **Timeline view** — memories arranged in chronological order, side by side
- **By Feeling view** — memories grouped by mood: Joy, Love, Adventure, Cozy, Silly, Tender, Milestone
- **Search** — find any memory by title, story, or tag
- **Filter by mood** — quickly narrow down to moments that match a certain feeling
- **Photo upload** — drag & drop or click to upload; stored locally in your browser
- **Lightbox** — click any memory to open a full-screen view with all its details
- **Add / Edit / Delete** — full control over every memory
- **Keyboard shortcuts** — press `Esc` to close any open panel

| Layer      | What's used           |

| Styling    | CSS3 (custom properties, grid, flexbox, animations) |
| Logic      | Vanilla JavaScript (ES6+) |
| Storage    | Browser `localStorage` |
| Fonts      | Cormorant Garamond + DM Sans (Google Fonts) |

## Privacy

Everything lives in your browser's local storage. No data is sent anywhere. No backend. No accounts. Just you two.
